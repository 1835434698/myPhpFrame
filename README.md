# myPhpFrame

#### 项目介绍
框架

#### 软件架构
这是一个php项目框架，框架本身已经实现了管理员系统、基础数据库、api接口系统、api日志保存与查看系统、常用工具等。


#### 安装教程

1. 拷贝代码。
2. 创建数据库执行sql语句。
3. 配置数据库username和password。
4.运行php服务。

#### 使用说明

1. api接口有示例有文档。
